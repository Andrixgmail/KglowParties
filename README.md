# KglowParties

Welcome to **KglowParties** – Your ultimate destination for unforgettable glow-themed events and parties!

## 🌟 About

KglowParties is a vibrant platform dedicated to hosting, organizing, and promoting exciting glow-in-the-dark parties and neon-themed events. Whether you're looking to discover the hottest glow events in your area or create your own luminous celebration, KglowParties is here to make it happen!

## ✨ Features

- **Event Discovery** – Browse and find glow parties happening near you
- **Event Creation** – Easily organize and promote your own glow-themed events
- **Community** – Connect with other party enthusiasts and event organizers
- **Event Details** – Complete information including dates, times, locations, and ticket information
- **Responsive Design** – Enjoy seamless browsing on desktop, tablet, and mobile devices

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required for static HTML

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Andrixgmail/KglowParties.git
   ```

2. Navigate to the project directory:
   ```bash
   cd KglowParties
   ```

3. Open `index.html` in your web browser:
   ```bash
   open index.html
   ```

   Or simply double-click the `index.html` file to open it in your default browser.
